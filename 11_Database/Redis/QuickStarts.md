## 安装

### Ubuntu/Debian

```bash
sudo apt-get update
sudo apt-get install redis
```

### Red Hat/Rocky

```bash
sudo yum install redis
sudo systemctl enable redis
sudo systemctl start redis
```

## 启动 / 关闭

```bash
sudo systemctl start redis
sudo systemctl stop redis
```

## running redis-cli

```
redis-cli
```
